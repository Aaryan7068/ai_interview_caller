AI Interview Caller

Design Document

Overall Architecture

The AI Interview Caller is a modern web application designed to streamline the initial candidate screening process. It features a decoupled architecture with a powerful backend and is ready for a dynamic frontend, ensuring scalability and maintainability.

  • Backend: Built with FastAPI, a high-performance Python web framework, the backend manages all core business logic, from handling API requests to database interactions and third-party service integrations.
  • Database: A relational database, likely PostgreSQL, is used for data persistence. SQLAlchemy serves as the Object-Relational Mapper (ORM), providing an efficient and Pythonic way to interact with the database.
  • AI Integration: At its core, the application integrates with a Large Language Model (LLM) service, such as Google's Gemini, to power its intelligent features. This integration is central to the application's value proposition.
  • Telephony Service: The system leverages Twilio to conduct automated phone interviews, managing call initiation, recording, and transcription services, which are crucial for the interview process.

Data Design Summary

The data model is designed for simplicity and efficiency, revolving around three primary tables that manage the entire interview lifecycle:

Table Purpose
JobDescription Stores details about job openings, including titles, content, and AI-generated interview questions.
Candidate Contains information about each candidate, such as their name, phone number, and a parsed summary of their resume.
InterviewResult Logs the outcomes of each interview, including the call SID, transcribed responses, scores, and the final hiring recommendation.

AI Usage Points

The AI Interview Caller utilizes a Large Language Model (LLM) to automate and enhance key recruitment tasks, making the screening process faster and more effective:

Question Generation

The LLM analyzes job descriptions to generate relevant and insightful interview questions, ensuring each interview is tailored to the specific role.

Resume Parsing

Upon application, the LLM parses candidate resumes to extract key information like skills and experience, creating a structured summary for easy review.

Response Scoring

The LLM analyzes transcribed interview responses to score them for quality and relevance, providing a data-driven hiring recommendation.
